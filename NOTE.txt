&conda常见命令：
	1.创建环境：conda create -n <env_name> python=X.X
	2.激活环境：activate <env_name>
	3.退出环境：deactivate
	4.删除环境：conda remove -n <env_name>
	5.复制环境：conda create --name <new_n> --clone <old_n>
	6.查询环境：conda env list 或 codda info --envs
	7.查询包：conda list
	8.更新conda：conda update conda
	9.查看环境详细信息：conda info
	10.查看镜像源：conda config --show channels
	11.添加镜像源：conda config --add channels <源链接>
